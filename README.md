# 🚀 AI Resume Analyzer

A simple AI-powered web app that analyzes resumes and provides feedback including score, strengths, weaknesses, and suggestions.

## 🔥 Features

* Upload PDF resumes
* AI-based analysis using Gemini API
* Score + feedback generation
* FastAPI backend

## 🛠️ Tech Stack

* Python
* FastAPI
* Google Gemini API
* PyPDF2

## ▶️ How to Run

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

Then open:
http://127.0.0.1:8000/docs

## ⚠️ Note

* Only PDF files supported
* API key required (not included for security)
